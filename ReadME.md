### Bootstrap 4.6
- CSS framework
- modules (collection of functions and classes)
- package (collection of modules)
- Library (collection of packages)
- Framework (collection of libraries)
+ module --> package --> library --> framework

+ type of modes
    - offline (getbootstrap.com)
    - online 
        - CDN links
+ background-color - bg
+ text-white
+ margin in Bootstrap (m-* (0-5))
    - 0 --> 0 rem
    - 1 --> 0.25 rem (4px)
    - 2 --> 0.5 rem (8px)
    - 3 --> 1 rem (16px)
    - 4 --> 1.5 rem (24px)
    - 5 --> 3 rem (48px)
    - ml-8 (margin left)
    - m (margin in all directions)
    - mt (margin-top)
    - mb (margin-bottom)
    - mr (margin-right)
+ padding (padding-left --> pl-5)

+ colors
    - primary
    - secondary
    - info
    - success
    - warning
    - danger
    - light
    - dark
    - white

+ We can use color classes for 
    - Buttons (btn btn-primary)
    - text (text-white)
    - background (bg-secondary)
    - alerts (alert alert-primary)

+ Grid System
    -   col-sm-12
    - sm (small devices)
    - md (medium devices)
